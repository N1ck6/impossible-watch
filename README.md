# Word Clock

Часы в стиле **Word Clock** с текстовым и цифровым отображением времени.

## Возможности

- **Текстовый режим** — время отображается словами (AM/PM)
- **Цифровой режим** — отображение часов, минут или секунд буквами в виде цифр
- **12/24-часовой формат** — переключается в цифровом режиме
- **Перетаскивание** — окно перемещается за область циферблата (не за прозрачный фон)
- **Системный трей** — иконка в трее с пунктом выхода
- **Поверх всех окон** — всегда видимы
- **Без рамки окна** — минималистичный вид

## Управление (4 точки по углам)

| Кнопка | Режим | Действие |
|--------|-------|----------|
| **Левая верхняя** | Горит = текст | Переключение между текстом и цифрами |
| **Левая нижняя** | Горит = минуты | В цифровом режиме показывает минуты (гасит правую нижнюю) |
| **Правая верхняя** | Горит = 12ч | Переключение 12/24-часового формата |
| **Правая нижняя** | Горит = секунды | В цифровом режиме показывает секунды (гасит левую нижнюю) |

> В текстовом режиме нижние кнопки отключены и не горят.

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск из исходников

```bash
python main.py
```

## Сборка в EXE

### Через скрипт:
```bash
python build.py
```

### Вручную:
```bash
pyinstaller main.py --name WordClock --noconfirm --clean --windowed --onedir
```

Готовое приложение будет находиться в папке `dist/WordClock/`.  
Запускать файл `dist/WordClock/WordClock.exe`.
